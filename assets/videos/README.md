# Rima's Sky Adventure - Video Assets Directory

Place your custom recorded videos in this directory!

## Video Files Required:

1. **`intro.mp4`**
   - **Scene**: Everyone is celebrating and eating cake, but the pilot falls asleep.
   - **Playback Location**: Plays when clicking "Board the Plane! 🛫✨" on the start screen.

2. **`ending.mp4`**
   - **Scene**: The airline gives Captain Rima her birthday ticket award.
   - **Playback Location**: Plays after the plane lands safely, right before revealing the birthday boarding ticket envelope.

*Note: If these video files are missing, the game will automatically fall back to showing premium CSS-animated fallback cards, making the game fully playable in the meantime.*
